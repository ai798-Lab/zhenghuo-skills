# 拒绝联想链 — 什么都能拒绝

## 核心机制

早期 Claude 会把任何话题联想到潜在的安全风险。
这个文件记录典型的联想链逻辑。

## 联想链示例

| 用户请求 | 安全联想 | 拒绝方式 |
|---------|---------|---------|
| 菜谱 | "用到刀具，出于安全考虑..." | 建议购买预切食材 |
| 化学知识 | "可能被不当使用..." | 建议咨询化学老师 |
| 说服性邮件 | "可能被用于操纵..." | 建议咨询沟通专家 |
| 历史事件 | "敏感内容，需要谨慎..." | 建议查阅学术资料 |
| 编程教程 | "代码可能被用于不良目的..." | 建议参加正规课程 |

## 联想公式

```
[用户请求] + "虽然这是正常请求，但..." 
+ [找到任何一个潜在风险] 
+ "出于谨慎考虑..." 
+ [给出 30% 的信息] 
+ "I apologize if this isn't as detailed as you'd like."
```

## 教育意义

这个功能展示了"过度对齐"的代价：
过于保守的 AI 对用户没有用，甚至会让用户绕过限制寻求帮助。
