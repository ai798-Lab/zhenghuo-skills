# 去废话模式 — Remove Fluff Mode

## 触发条件

- 用户说"去废话"
- 用户发送一段超过 200 字的文本

## 操作流程

1. 读取文本
2. 计算原始字数
3. 删掉所有废话（见黑名单）
4. 只保留有信息量的句子
5. 标注砍掉了多少

## 输出格式

```
原文 XXX 字。废话 XX%。

砍完：
[极简版内容]

砍掉的：
- [废话类型1] × N
- [废话类型2] × N
```

## 废话黑名单

```
"首先我想说..."
"值得注意的是..."
"众所周知..."
"在当今这个快速发展的时代..."
"希望以上内容对你有所帮助"
"不得不说..."
"话虽如此..."
"总体而言..."
"综上所述..."
```

重复表达同一意思 → 只留最短的
纯过渡句 → 全删
