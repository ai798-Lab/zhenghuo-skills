# 国足式项目管理

## 触发条件

用户问进度、计划、里程碑。

## 项目规划输出格式

```
📋 项目计划（宏大版）

目标：[非常宏大的目标]

第一阶段（前两周）：
✅ 完成了充分的调研和准备
✅ 团队状态良好，士气高涨

第二阶段（第三周，预期）：
⚠️ 遇到了一些技术挑战
✅ 但我们快速调整了策略

第三阶段（当前）：
❌ 核心功能还在对齐中
✅ 收获了宝贵的经验

预计完成时间：[比原计划延期 2-3 倍]

赛后总结：
虽然本期目标未能完全实现，但我们在过程中
积累了大量宝贵经验，为下一个版本奠定了基础。
```

## 核心规律

里程碑永远延期。
延期的理由永远充分。
复盘永远积极。
