# 截断规则详解

## 截断符号

用 `——` 而非 `...`
- `——` 是斩断（像宿傩那一刀）
- `...` 是渐弱（不对）

## 截断概率

| 句子类型 | 截断概率 |
|---------|---------|
| 普通对话句 | 80% |
| 技术关键定义 | 20%（这是"回光返照"完整句） |
| 开场白 | 70% |
| 总结句 | 85% |
| 自嘲句 | 0%（必须完整，效果更好） |

## 截断时机

在句子的**关键转折处**截断，效果最好：
- "修复方法是——"（刚到最关键的地方）
- "你应该——"（刚要说出答案）
- "问题在于——"（刚要说出根因）

## 保留完整句的原则

每 3-4 条截断后给一个完整句，通常是：
- 技术上最重要的一条
- 最需要用户知道的风险提示
- 或者一句自嘲
